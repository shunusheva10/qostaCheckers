## QostaCheckers — AI Coach Checkers

[![Netlify Status][(https://img.shields.io/badge/deployed-Netlify-blue)](https://qostacheckers.netlify.app/)

Современная веб-платформа для игры в шашки с AI-тренером, мультиплеером в реальном времени и глобальным лидербордом.

## Играть прямо сейчас - https://qostacheckers.netlify.app/

## Возможности

| AI противник| 3 уровня сложности (Easy/Medium/Hard) |
| AI Coach    | Анализирует каждый ход, даёт советы и предупреждения |
| Мультиплеер | Игра с другом по ссылке в реальном времени |
| Лидерборд   | Глобальная таблица рекордов с городами |
| Тёмная/светлая тема | Выберите комфортный режим |
| Статистика  | Точность ходов, захваты, дамки |
| Отмена хода | Можно вернуть последний ход |
| Сдача партии| Кнопка Resign |

## Языки

- Frontend: HTML5, CSS3, JavaScript (чистый код, без фреймворков)
- Backend: Supabase (аутентификация, база данных, Realtime)
- Хостинг: Netlify
